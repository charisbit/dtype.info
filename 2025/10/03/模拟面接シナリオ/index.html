① 日本にほん語ご 自己じこ紹介しょうかい（2-3分ふん）

はじめまして、盛と申しもうします。
中国ちゅうごくの大学院だいがくいんでシステムしすてむ分析ぶんせきを学びまなび、2016年ねんから日本にっぽんでITエンジニアえんじにあとして働いてはたらいてきました。
Pythonを中心ちゅうしんに、バックエンドからインフラいんふらまで幅広くはばひろく経験けいけんしています。

直近ちょっきんのプロジェクトぷろじぇくとでは、FastAPIを使ったつかったAIアプリケーションあぷりけーしょんの開発かいはつや、pytestでの自動じどうテストてすとを実装じっそうしました。
SQLAlchemyを利用りようしたORMの設計せっけいや、PostgreSQLでのデータベースでーたべーす構築こうちくも担当たんとうしました。
シェルスクリプトすくりぷとを書いてかいて、CI/CDパイプぱいぷラインらいんや運用うんようの自動じどう化かにも対応たいおうしています。

また、生成せいせいAIとRAGの仕組みしくみを活用かつようして、社しゃ内ないエージェントや監視かんし自動じどう化かシステムしすてむを作ったつくった経験けいけんがあります。
AzureやAWSを使ったつかったクラウド環境かんきょうの構築こうちくも行いおこない、VM、Blobストレージ、ネットワークねっとわーく設計せっけいを担当たんとうしました。

日本にほん語ご環境かんきょうで長くながく仕事しごとをしてきましたので、コミュニケーションこみゅにけーしょんにも問題もんだいありません。
これまでの経験けいけんを活かしていかして、今回こんかいの案件あんけんにしっかり貢献こうけんできると考えてかんがえています。
どうぞよろしくお願いねがいいたします。

想定そうてい質問しつもんと回答かいとう（日文にちもん）

Q1: FastAPIを使ったつかった経験けいけんは具体ぐたい的にてきにどんなものですか？
A1: AIアプリケーションあぷりけーしょんでFastAPIを使いつかい、REST APIを実装じっそうしました。SwaggerでAPI仕様しようをまとめ、pytestでテストてすとしました。

Q2: RAGを使ったつかった開発かいはつ経験けいけんを教えておしえてください。
A2: AWSとLangChainを利用りようして、RAG構成こうせいを作りつくりました。知識ちしきベースべーすと大だい規模きぼ言語げんごモデルもでるを統合とうごうし、検索けんさくと回答かいとうを自動じどう化かしました。

Q3: Azureの経験けいけんはありますか？
A3: はい。Azure VMやBlobストレージを構築こうちくしました。Azure Container Registryも利用りようして、AIサービスさーびすをデプロイしました。

逆ぎゃく質問しつもん（日文にちもん）

  1. 今回こんかいのプロジェクトぷろじぇくとでは、生成せいせいAIとRAGをどのようなユースケースで使うつかう予定よていですか？
  2. チームちーむの開発かいはつ体制たいせいや人数にんずうはどのくらいですか？
  3. 出社しゅっしゃとリモートの比率ひりつは週しゅうごとに固定こていされていますか？

③ プロジェクトぷろじぇくと紹介しょうかい（案件あんけん要件ようけんに合うあうもの）

日本にほん語ご

私わたしが今回こんかいの案件あんけんに一番いちばん近いちかい経験けいけんは、**社しゃ内ないエージェントシステムしすてむ『Intelligent Agent Platform』** の開発かいはつです。
ここでは、PythonとFastAPIを使ってつかってサーバーレスのAIエージェントを実装じっそうしました。
SQLAlchemyでデータでーた管理かんりを行いおこない、pytestでテストてすとを自動じどう化かしました。
データベースでーたべーすはPostgreSQLを利用りようしました。
また、Shellスクリプトすくりぷとを使ってつかって、デプロイとTerraformレビューれびゅーを自動じどう化かしました。

AIの部分ぶぶんでは、RAG構成こうせいを使いつかい、LangChainとAWS Bedrock、Kendraを統合とうごうしました。
この経験けいけんは、生成せいせいAIやRAGを利用りようしたアプリあぷり開発かいはつに直接ちょくせつつながります。
さらに、Azure VMやBlobストレージを利用りようしたシステムしすてむ構築こうちく経験けいけんもあるため、今回こんかいの案件あんけん要件ようけんにマッチまっちしていると思いおもいます。

英語えいご

The most relevant project is the “Intelligent Agent Platform” I developed.
In this project, I used Python and FastAPI to build a serverless AI agent system.
I managed data with SQLAlchemy and wrote automated tests with pytest.
The database was PostgreSQL.
I also wrote shell scripts for deployment and Terraform reviews.

For the AI part, I implemented a RAG system using LangChain, AWS Bedrock, and Kendra.
This is directly related to Generative AI and RAG-based application development.
In addition, I have experience with Azure VM and Blob storage setup, so I believe my background fits well with the requirements of this project.
